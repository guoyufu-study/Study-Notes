# pip3

> 安装和管理 Python3 软件包

``` shell
# python39-pip.noarch : A tool for installing and managing Python3 packages
yum install python39-pip
```

> 依赖关系：
>
> python39、python39-libs、python39-pip-wheel、python39-setuptools-wheel
>
> 弱依赖：
>
> python39-setuptools 
>
> 启用模块流：
>
> python39
