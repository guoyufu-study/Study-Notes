MySQL 服务器 ( [**mysqld**](https://dev.mysql.com/doc/refman/8.0/en/mysqld.html) ) 是在 MySQL 安装中完成大部分工作的主程序。本章提供 MySQL 服务器的概述并涵盖一般服务器管理：

- 服务器配置
- 数据目录，尤其是`mysql` 系统模式
- 服务器日志文件
-  在一台机器上管理多台服务器
