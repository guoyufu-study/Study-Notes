适者生存：编程语言生态系统的气候不断变化，新语言会涌现，旧语言会消亡，墨守成规就会被取代。完美的通用语言不存在，现实中某些语言只是更适合某些方面。

> 气候变化：新硬件或新编程因素，意味眷新项目优先的语言各有不同。
>
> 新语言不断涌现，并因为迅速适应了气候变化，越来越受欢迎。

抢占先机：先抢占市场往往能够吓退竞争对手。
