Scala 将面向对象编程和函数式编程结合在一种简洁的高级语言中。Scala 的静态类型有助于避免复杂应用中的 bug，它的 JVM 和 JavaScript 运行时允许构建高性能系统，从而方便地访问庞大的库生态系统。
