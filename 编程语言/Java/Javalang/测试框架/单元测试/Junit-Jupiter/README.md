# JUnit 5

> https://junit.org/junit5/

JUnit 5 是下一代 JUnit。 目标是为 JVM 上的开发人员测试创建一个最新的基础。 这包括专注于 Java 8 及更高版本，以及启用许多不同风格的测试。
